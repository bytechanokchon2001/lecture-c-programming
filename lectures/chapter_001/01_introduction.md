# Introduction
เพื่อให้เราเขียนโปรแกรมได้เร็วที่สุด ในบทนี้เราจะมุ่งเน้นไปที่ 
- ตัวแปรและค่าคงที่
- การคำนวนทางคณิตศาสตร์
- การควบคุมการไหลของโปรแกรม
- ฟังก์ชัน
- พื้นฐานการรับและส่งข้อมูล

ธรรมเนียมของการเรียนรู้ภาษาโปรแกรมใหม่ เราจะต้องแสดงข้อความ `Hello World` โดยใน C สามารถทำได้ดังนี้

    #include <stdio.h>

    int main() {
        printf("Hello World");

        return 0;
    }

โดยวิธีการเรียกใช้งานโปรแกรมจะขึ้นอยู่กับระบบปฏิบัติการ สำหรับ windows สามารถใช้งานคำสั่ง `gcc <file_name>.c` เมื่อใช้คำสั่งแล้วจะได้รับไฟล์ `.exe` มา เราสามารถเรียกใช้งานไฟล์ดังกล่าวเมื่อให้โปรแกรมทำงานได้ทันที

จากชุดคำสั่งด้านบน `main` คือฟังก์ชันแรกของโปรแกรมที่จะถูกเรียกใช้งาน เมื่อโปรแกรมถูกสั่งให้ทำงาน โดยเราจะใช้ main ในการเรียกฟังก์ชันที่เราเขียนเองหรือฟังก์ชันที่มาจากไลบรารี่ภายนอก

`#include<stdio.h>` เป็นการนำชุดคำสั่งจากไฟล์ที่ระบุ มาใส่ไว้ในจุดที่ชุดคำสั่งนี้ถูกวางไว้ โดยในขึ้นตอนการนำชุดคำสั่งมาใส่จะเกิดขึ้นขนะคอมไพล์ โดยในตัวอย่างคือการนำชุดคำสั่งที่เกี่ยวกับ input/output มาใส่

ในการส่งข้อมูลให้กับฟังก์ชัน เราสามารถส่งข้อมูลเข้าไประหว่าง `()` ของฟังก์ชันได้ โดยข้อมูลที่ถูกส่งเข้าไปจะถูกเรียกว่า **Argrement**

    #include<stdio.h>               // 1

    main() {                        // 2
        printf("Hello World");      // 3
    }

ึคำอธิบาย
1. นำเข้าชุดคำสั่งจาก standard library
2. ประกาศฟังก์ชัน main โดยไม่รับค่าใด ๆ ทาง argrement โดยคำสั่งของฟังก์ชันจะอยู่ภายใต้ {}
3. ฟังก์ชัน main เรียกใช้ฟังก์ชัน printf ที่มาจาก stdio.h เพื่อแสดงข้อความ Hello World ออกไปที่หน้าจอ โดย `printf` จะลงท้ายด้วย `\n` เพื่อขึ้นบรรทัดใหม่เสมอ
4. "Hello World" เป็นชุดลำดับของอักขระเรียกว่า สตริง หรือ ค่าคงที่สตริง โดยในสตริงหากต้องการขึ้นบรรทัดใหม่ จะต้องใช้คำสั่ง `\n` เท่านั้น
